SMH | Root Page (v1)
====
Extensible Root Page component written in HTL.
This page component will be used to define the root node of the website directly under `/content` node.
This page can be overridden to have site admin related configurations.

## Information
* **Vendor**: SMH
* **Version**: v1
* **Compatibility**: AEM 6.5.11
* **Status**: production-ready
